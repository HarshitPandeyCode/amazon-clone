# 🛒 Amazon Clone

A simple Amazon homepage clone built using **HTML5** and **CSS3**. This project focuses on recreating the layout and design of Amazon's landing page while practicing core front-end development concepts.

## ✨ Features

* Responsive navigation bar
* Search box and menu section
* Hero banner
* Product category cards
* Footer section
* Clean and organized code structure

## 🛠️ Built With

* HTML5
* CSS3

## 🎯 Project Goal

The goal of this project was to improve my understanding of HTML and CSS by recreating a real-world website interface and strengthening my skills in page layout, styling, and responsive design.
